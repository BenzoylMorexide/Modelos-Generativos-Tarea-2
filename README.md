# Modelos Generativos - Tarea 2

Este repositorio contiene la implementación y los resultados de la **Tarea 2** de la asignatura *Modelos Generativos*. El proyecto incluye el código de ejecución, métricas de evaluación y la generación de imágenes.

## 📁 Estructura del Repositorio

El proyecto se organiza de la siguiente manera:

```text
.
├── Exploracion_parte_1a/      # Contiene las 10 imágenes generadas en la primera parte de la tarea.
├── Imágenes_y_métricas/       # Contiene:
│   ├── metrics.csv            # Archivo con las métricas de evaluación (calculadas mediante el notebook).
│   └── generated_images/      # Las 45 imágenes finales generadas.
├── tarea2.ipynb               # Notebook principal con el código de implementación y ejecución.
└── README.md                  # Documentación del proyecto.
